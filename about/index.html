本站

前段时间尝试用 Express + Vue3 搭建了一个摄影知识网站，也算是带有社区功能的博客。众所周知：“造不如买，买不如······”，一顿折腾之后发现 Hexo 这种静态博客特别适合我，可以一次生成、随处部署。暂时白嫖一下 Github Page，日后再购入自己的域名。
