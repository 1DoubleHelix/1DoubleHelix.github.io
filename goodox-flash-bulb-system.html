神牛闪光系统产品线解析

本文并非专业的选购指导，购买产品前请咨询官方客服以获取详细描述

一些主要的功能标签：无线ID​​ 镍氢电池​​ 锂电池​​ TTL​​ 点阵屏​​

在了解具体的产品前，需要关注以下几点

  1. 预算，没钱万万不能
  2. 只作机顶灯使用：不想手动调光，关注 TTL​​​ 功能
  3. 无线环境复杂：例如漫展场内，关注 无线ID​​​ 功能
  4. 长时间拍摄大量图片：考虑 锂电池​​​ 灯，甚至外拍灯，亦或者常亮灯
  5. 如果选择了使用镍氢电池的产品，可以参考电池技术来选购电池，镍氢电池​​​ 有自身的优点，但是充电慢、能量小的缺点有点廉颇老矣的意思

机顶（热靴）灯

  • 神牛的机顶灯型号很多，初看可能难以分别，根据使用经验可以简单分为五个模具
  • 除了旗舰型号 V1 和非常入门的 TT520/TT560 系列，其他三个模具可以看作是电源类型不同的版本
  • V（逸客）系列为锂电、TT（迅丽）系列为镍氢电池

​​

V1 无线ID​​ 锂电池​​ TTL​​ 点阵屏​​

​​

  • 旗舰机顶灯，不缺钱首选
  • 电池和 AD100Pro 通用
  • 需要购买对应的热靴型号

V860II/TT685 无线ID​ 镍氢电池​ 锂电池​ TTL​ 点阵屏​

​​​

  • 这个模具的特点是支持 TTL 测光，点阵屏幕
  • 作为离机灯使用的时候，支持无线 ID（V860 仅二代以后型号）在漫展神牛无线设备很多的情况下，可以几乎不受影响地拍摄
  • V860II 升级到 1.4 固件后支持无线 ID
  • 需要购买对应的热靴型号

V850II/TT600 镍氢电池​ 锂电池​ 无线ID​

​​

  • 这个模具都是手动灯，显示屏比较简单，仅 V850III 支持无线 ID⚠
  • 不支持无线 ID 的灯在漫展场内拍摄时被他人引闪干扰概率很高，十分影响拍摄，但是只作为机顶灯或者棚拍等无线环境比较干净的应用场景，值得考虑
  • TTL 测光和高速同步功能需要配合引闪器使用，
  • 除了索尼建议购买 TT600S外，其他品牌通用
  • V850 第一代没有内置无线接收器，不支持无线引闪，捡垃圾注意⚠

V350/TT350 TTL​ 无线ID​

​​​

  • 比较新的产品，主打小巧的同时功率也一般，支持 TTL 测光
  • 仅 V350 支持无线 ID，镍氢版本不支持
  • 需要购买对应的热靴型号

TT520/TT560 镍氢电池​

​​​

  • 这个面板就十分的复古，手动灯
  • 离机引闪需要拨码，如果不是预算十分紧缺不推荐用这个干活
  • 玩耍随意

外拍灯

厂商标注的输出功率不是严格的 W（瓦），而是 Ws（实际为焦耳），即单次闪光释放的最大能量，常亮灯标注的才是 W（瓦）

  • 神牛外拍灯的命名非常规则，AD + 输出功率，根据自己的需求选择即可，这里只简单介绍一下 AD200 和 AD100Pro，外拍灯都有无线 ID，这个放心
  • 找到一个非常有意思的对比视频（伪），如果你在纠结是否要入手 AD100Pro，可以看看 Godox Flashpoint Xplor AD100 vs AD200 vs AD300 vs AD400 vs AD600 vs ?

AD200/Pro 无线ID​ 锂电池​ TTL​

​​​

  • 神器，兼顾了体积和输出功率
  • 虽然比机顶灯重一点，但带来的是 200Ws 的输出功率

AD100Pro 无线ID​ 锂电池​ TTL​ 电动变焦​

​​​

  • 一罐可乐大小的新成员
  • 100Ws 输出，和 V1 的电池通用

引闪系统兼容性

  • X1、X2、XPro 是新一代引闪，XT32 是旧款引闪，无线闪光系统兼容性如下图
  • 如果想要极致地省钱，购买第一代 V850、V860，但是这两个型号没有内置无线接收，所以并不建议购买第一代锂电灯

​​

引闪器

  • 神牛无线引闪器除以下常见的型号外，还有旧款的 XT32，大概率用不上
  • 引闪的主要功能都差不多，最早的 X1 也支持无线 ID
  • 如果想调参方便就买 X2、XPro
  • 请使用对应相机品牌的版本
